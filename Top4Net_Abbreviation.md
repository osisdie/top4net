## 缩写约定 ##
| **缩写** | **全称** | **描述** |
|:-----------|:-----------|:-----------|
| cid | Category Id | 类目编号 |
| iid | Item Id | 商品编号 |
| oid | Order Id | 订单编号 |
| pid | Property Id | 属性编号/阿里妈妈编号 |
| rid | Refund Id | 退款编号 |
| sid | Shop/Shipping Id | 店铺/运单编号 |
| tid | Trade Id | 交易编号 |
| uid | User Id | 用户编号 |
| vid | Value Id | 属性值编号 |

注：商品编号目前统一用：product\_id

## 其它约定 ##
  * Id: 表示编号
  * No: 表示账号（如支付宝账号：alipay\_no）
  * Num: 表示数量
